这两个键的位置实在太恶心了，在中文输入这样经常敲击 ctrl 的情况下面。很容易造成在输入 blog 或者发帖子的时候被   back 掉，好不容易写了半天的文章就这么被废掉了。

禁止方式：

开始菜单   -> Access IBM -> Keyboard  Customizer Utility -> Key Sensivity -> Enable Browser Keys 去掉
